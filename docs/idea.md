# Идея проекта

Веб-приложение для проверки сайтов на соответствие законодательству РФ в части, касающейся сайтов и веб-приложений.

## Для кого

- Владельцы сайтов и интернет-магазинов.
- Маркетологи и продакт-менеджеры, отвечающие за лендинги и продукты.
- Веб-студии и фрилансеры, сдающие сайты клиентам.
- Юристы и комплаенс-специалисты, проверяющие чужие ресурсы.

## Какую задачу решает

Снимает с пользователя необходимость вручную сверять сайт с разрозненными требованиями российского законодательства (152-ФЗ о персональных данных, требования к согласиям и cookie, обязательные сведения о продавце, маркировка рекламы и т. д.). Помогает заранее увидеть нарушения и оценить риски штрафов до того, как их найдёт регулятор.

## Доступ

Базовая проверка — бесплатная, доступна любому посетителю сайта без регистрации и без ограничения на количество запусков. В отчёт попадают все нарушения, которые удаётся определить по чётким признакам страницы (формы, cookie, обязательные блоки, маркировка рекламы).

Расширенный анализ — для авторизованного владельца. Над теми же страницами дополнительно работают семантические проверки через автоматический анализ текста (читаемость политики, корректность согласия, dark pattern в баннере и т. п.). Эти проверки идут через платные внешние сервисы, поэтому открыты только после входа.

В MVP — один пользователь-владелец, заводится владельцем сервера через CLI. Регистрации, восстановления пароля, личного кабинета на сайте нет.

## Сценарии пользы в первой версии

Пользователь вводит URL сайта в единственное поле на главной. Запускается проверка с анимацией, показывающей, что именно сейчас анализируется. По завершении пользователь получает отчёт:

- список найденных нарушений;
- где конкретно на сайте они обнаружены;
- какие штрафы предусмотрены за каждое нарушение;
- рекомендации, как исправить.

Отчёт можно скачать в PDF.

Типовые сценарии:

- **Быстрый аудит перед запуском** — проверить новый лендинг или сайт до публикации. Базовой проверки без входа достаточно для большинства типовых ошибок.
- **Проверка чужого сайта** — оценить ресурс клиента, подрядчика или конкурента. Бесплатно, без входа.
- **Аргумент для заказчика или руководства** — выгрузить PDF и показать конкретные риски и суммы штрафов. Для весомых формулировок (читаемость политики, корректность согласия) полезен расширенный анализ под входом.
- **Самопроверка после изменений** — убедиться, что правки на сайте не сломали соответствие закону. Бесплатно, без входа.

В MVP результаты проверок нигде не сохраняются — каждая проверка независима.
